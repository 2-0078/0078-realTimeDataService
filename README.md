## 🎯 프로젝트 개요

Real-Time Data Service는 **한국투자증권 API 연동**과 **실시간 시세 스트리밍**을 담당하는 서비스입니다.

### 핵심 기능
- **한국투자증권 API 연동**: 실시간 주식 시세 수신
- **SSE 시세 스트리밍**: 실시간 호가 및 체결가 스트리밍
- **차트 데이터 제공**: 분봉, 일봉, 기간별 시세 제공
- **WebSocket 연결**: 실시간 데이터 수신 및 처리
- **데이터 저장**: MongoDB 기반 시계열 데이터 저장

### 데이터 소스
- **한국투자증권 API**: 국내 주식 시세 데이터
- **실시간 호가**: 매수/매도 호가 정보
- **체결 정보**: 실시간 체결가 및 거래량
- **기술 지표**: 이동평균, RSI, MACD 등

## 🛠️ 기술 스택

### Backend
- **Java 17** - LTS 버전
- **Spring Boot 3.5.0** - 최신 프레임워크
- **Spring WebFlux** - 반응형 웹 스택
- **Spring WebSocket** - WebSocket 연결 관리
- **Spring Data MongoDB** - NoSQL 데이터베이스
- **Spring Data MongoDB Reactive** - 반응형 MongoDB

### Database & Cache
- **MongoDB 4.4** - 시계열 데이터 저장
- **Redis 7.0** - 실시간 데이터 캐싱
- **MySQL 8.0** - 메타데이터 저장

### Message Queue
- **Apache Kafka 3.0** - 실시간 데이터 스트리밍
- **Spring Kafka** - Kafka 연동

### External APIs
- **한국투자증권 API** - 실시간 주식 데이터
- **WebSocket** - 실시간 데이터 수신

### Service Discovery
- **Netflix Eureka** - 서비스 등록/발견

## 🚀 주요 기능

### 1. 실시간 시세 수신
- **WebSocket 연결**: 한국투자증권 API WebSocket 연결
- **실시간 호가**: 10단계 매수/매도 호가 수신
- **체결 정보**: 실시간 체결가, 거래량 수신
- **시세 변동**: 가격 변동률, 등락폭 계산

### 2. 시세 스트리밍
- **SSE 스트리밍**: 클라이언트로 실시간 시세 전송
- **다중 채널**: 여러 종목 동시 스트리밍 지원
- **연결 관리**: 연결 상태 모니터링 및 재연결
- **데이터 압축**: 효율적인 데이터 전송

### 3. 차트 데이터 제공
- **분봉 데이터**: 1분, 5분, 15분, 30분 봉 데이터
- **일봉 데이터**: 일별 OHLCV 데이터
- **기간별 조회**: 특정 기간 시세 데이터 조회
- **기술 지표**: 이동평균, 볼린저밴드, RSI 등

### 4. 데이터 저장 및 관리
- **시계열 데이터**: MongoDB 기반 시계열 데이터 저장
- **데이터 압축**: 효율적인 저장 공간 활용
- **자동 삭제**: 오래된 데이터 자동 삭제
- **백업 및 복구**: 데이터 백업 및 복구 시스템

## 📊 실시간 데이터 처리

### 데이터 처리 흐름
1. **한국투자증권 API WebSocket**: 실시간 데이터 수신
2. **데이터 파싱 및 검증**: 수신된 데이터 검증 및 변환
3. **MongoDB 저장**: 시계열 데이터 저장
4. **Redis 캐싱**: 실시간 데이터 캐싱
5. **SSE 스트리밍**: 클라이언트로 실시간 전송

### WebSocket 데이터 처리 특징
- **연결 관리**: 안정적인 WebSocket 연결 유지
- **재연결 로직**: 연결 끊김 시 자동 재연결
- **데이터 검증**: 수신 데이터 유효성 검증
- **에러 처리**: 데이터 수신 실패 시 적절한 에러 처리

### 실시간 스트리밍 처리
- **SSE 연결 관리**: 다중 클라이언트 연결 관리
- **실시간 전송**: 지연 없는 데이터 전송
- **선택적 구독**: 특정 종목만 선택적 구독
- **연결 상태 모니터링**: 클라이언트 연결 상태 추적

## ⚙️ 환경 설정

### 필수 요구사항
- Java 17 이상
- MongoDB 4.4 이상
- Redis 7.0 이상
- 한국투자증권 API 키
- Docker (선택사항)

### 환경 변수
- **MONGODB_URI**: MongoDB 연결 정보
- **REDIS_URL**: Redis 서버 연결 정보
- **KIS_API_KEY**: 한국투자증권 API 키
- **KIS_API_SECRET**: 한국투자증권 API 시크릿
- **KAFKA_BOOTSTRAP_SERVERS**: Kafka 서버 정보
- **EUREKA_CLIENT_SERVICE_URL**: Eureka 서버 정보

## 📚 API 문서

### Swagger UI 접속
API 문서는 Swagger UI를 통해 확인할 수 있습니다.

### 주요 API 엔드포인트

#### 실시간 스트리밍
- **실시간 시세 SSE**: 특정 종목의 실시간 시세 스트리밍
- **실시간 호가 SSE**: 실시간 호가 정보 스트리밍
- **실시간 체결 SSE**: 실시간 체결 정보 스트리밍

#### 시세 조회
- **현재 시세 조회**: 특정 종목의 현재 시세 정보
- **과거 시세 조회**: 특정 기간의 시세 데이터
- **차트 데이터 조회**: 분봉, 일봉 차트 데이터

#### 호가 정보
- **호가 정보 조회**: 특정 종목의 호가 정보
- **호가 깊이 조회**: 상세 호가 정보

#### 기술 지표
- **기술 지표 조회**: 이동평균, RSI, MACD 등
- **이동평균 조회**: 특정 기간 이동평균
- **RSI 조회**: 상대강도지수
